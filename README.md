# MH Spear Master (MH武器大師) 🔱

這是一款專為漁獵（Spearfishing）愛好者設計的**實戰魚槍配置與彈道解算工具**。
無論是傳統皮筋、滾輪（Roller）還是逆變系統（Invert），本工具都能進行精準的物理推演。

🌍 **線上立即使用**：[點擊這裡進入系統](https://你的帳號名.github.io/mh-spear-master/) (請替換為你的真實網址)

## ✨ 核心功能
* **雙向物理解算**：支持正向導購配置，也支持「現貨皮筋長度」的逆向拉伸比反推。
* **五大核心指數雷達圖**：裝填、後坐力、擺動性、浮力、便携性，一眼看穿配置優劣。
* **自動配重 (Ballast)**：根據槍身材質體積，自動計算木槍/碳纖維的中性懸浮配平重量。
* **彈道下墜預測**：結合水阻與槍頭倒刺升力，直觀展示 1~10 米的動能與下墜軌跡。
* **多國語言支援**：中、英、日等 7 國語言無縫切換。
* **配置單導出**：一鍵生成包含雷達圖的高清配置單，支援手機端長按儲存。

## 🚀 技術堆疊
* 純前端單頁應用 (HTML5 / JS / CSS3)
* 不依賴任何後端伺服器，支援離線/弱網環境計算。
* 內建 `html2canvas` 實現客戶端高清截圖。
